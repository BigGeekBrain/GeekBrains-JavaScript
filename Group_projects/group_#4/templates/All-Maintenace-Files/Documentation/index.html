A) HTML Structure - top

The template is made with responsive layout with Bootstrap framework. All the content is within section. All the part of the template is divided with section. Main infrastructure is same into the all section. All contents are rounded with div, id and class. Most of the class and id are used according to Bootstrap. Here is the general structure.

C) JavaScript - top

This theme imports three Javascript files.

  1. jQuery
  2. Bootstrap
  3. Countdown
  4. YTPlayer
  5. Backstretch
  6. Scripts
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. Bootstrap is a web framework.
  3. Countdown is a jQuery plugin for counting time.
  4. YTPlayer is a jQuery plugin for youtube background video.
  5. Backstretch is a jQuery plugin for slider background.
  6. There are some customizable Scripts, These Scripts are changeable as per your demand.
  • How to edit countdown time
    • Open "js/scripts.js" File and edit this time. 
    ----- Countdown -----
			
"January 26, 2016 20:39:00"
  • How to add slider
    • Open "js/scripts.js" File and edit this section. 
    ----- Backstretch Slider -----
			
  "images/slider1.jpg"
, "images/slider2.jpg"
, "images/slider3.jpg"

    • Open "index.html" File and edit this section. 
    ----- Youtube Video -----

  • How to add MailChimp URL
